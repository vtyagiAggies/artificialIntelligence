Project is developed in Python. 


In change directory to the folder and execute following command:
./search_main.py arg0 arg1 arg2 arg3 agr4 arg5 agr6

where 
arg0: path to graph file 
arg1: type of search algorithm in format ("BFS", "DFS", "GBFS", "ASTAR")
arg2, arg3: Initial state coordinate 
arg4, arg5: Destination coordinate 
arg6 (optional): for debug (--debug)

example: ./search_main.py ATM.graph BFS 1 20 20 1
