How to run the program

Project is developed in Python. 

Steps:
1. Change directory to root of the project folder
2. make proposition_main.py exectable by command:
	chmod +x proposition_main.py 
3. Exectue proposition_main.py by
	./proposition_main.py arg0 arg1

where 
arg0: path to knowledge base file (default: input.txt)
arg1 (optional): for debug (--debug)

example: ./proposition_main.py test.txt --debug
